# Mashiro

> 基于go-cqhttp，由Python开发的QQ机器人.

* 完整的go-cqhttp功能：通过Websocket与go-cqhttp服务端连接
* 快速部署：自带安装脚本，一条命令完成安装
* 简单的插件开发：基于Python，并配备完善的接口，简化QQ机器人插件开发
